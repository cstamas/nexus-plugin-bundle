Class Overview

Supplemental configuration for plugin bundle assemblies, to allow users to customize the finalName, archive configuration (the MavenArchiveConfiguration instance), and tarLongFileMode. This class is similar to the plugin parameters available in the maven-assembly-plugin itself.
